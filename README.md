# IICT-assginment-2
Website for assignment
## Project Overview
This is a single-page responsive mini-site created for **Assignment 02**. The fictional product is the **AB-01 Interdimensional Portal Device** by **Abdullah Science**. The site uses **HTML5 + CSS3 only** (no JavaScript) and demonstrates:

- Semantic HTML structure 
- Three breakpoints: **mobile / tablet / desktop**
- A fully functional and styled HTML form (text inputs, checkboxes, radio buttons, select, textarea, submit)
- **three CSS-only interactive components** and **Flexbox**:
  - Tabs (radio button hack)
  - Accordion (checkbox hack)
  - Modal (checkbox hack)
- Clean CSS variables for easy editing


  AUTHOR'S NOTE: Made in total accordance with the assignment, nothing is real, proper labeling and easily editable
   (do not touch the image design or else everything will be broken)
